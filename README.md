# TAVARDT Elite Boilerplate

> A High-Performance Next.js Boilerplate achieving 100/100 PageSpeed scores, tailored for the Elite Digital Market.

![100/100 PageSpeed](https://img.shields.io/badge/PageSpeed-100%2F100-brightgreen?style=for-the-badge)
![Next.js](https://img.shields.io/badge/Next.js-15-black?style=for-the-badge&logo=next.js)
![Tailwind CSS](https://img.shields.io/badge/Tailwind_CSS-3.4-38B2AC?style=for-the-badge&logo=tailwind-css)

## ⚡ Why this boilerplate?

At **TAVARDT**, we build invisible brands and transform them into market authorities. This boilerplate is our proprietary foundation designed to deliver:
- **Absolute Performance:** Critical Rendering Path optimization, preloaded fonts, and rigorous lazy loading.
- **"Elite Dark Glassmorphism" Aesthetics:** Pre-configured design system in Tailwind CSS.
- **SEO Ready:** Structured metadata and embedded JSON-LD.

## 🚀 How to use

```bash
# Clone the repository
git clone https://github.com/TAVARDT/tavardt-elite-boilerplate.git

# Install dependencies
npm install

# Run the development server
npm run dev
```

## 🛠️ Tech Stack

- **Framework:** Next.js (App Router)
- **Styling:** Tailwind CSS + Vanilla CSS (for specific optimizations)
- **Animations:** Framer Motion (configured to load without penalizing hydration)
- **Deployment:** Optimized for Vercel or VPS (Docker/Nginx)

## 🔒 Implemented Best Practices
- No unnecessary global hydration.
- Images served in WebP/AVIF.
- Use of `next/font` to prevent Cumulative Layout Shift (CLS).
